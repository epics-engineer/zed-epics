# Changelog

## 0.1.0

Initial release.

- Add Zed language configs for EPICS command files, database files, MSI substitutions, MSI templates, SNL, and StreamDevice protocols.
- Register the split Tree-sitter grammar repositories from `minijackson`.
- Use upstream highlight and injection queries from the pinned grammar releases.
- Leave `.proto` unclaimed by default so Protocol Buffers files keep their usual language association.
